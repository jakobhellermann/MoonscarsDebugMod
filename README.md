# Development

1. Make sure the path to your Moonscars installation in `LocalBuildProperties.props` is correct.
2. Build the project
    ```sh
    dotnet build
    ```
3. Run the game

# Keyboard Shortcuts

- `O`: Exit to main menu
- `Alt+B`: Toggle Hitboxes
- `Alt+.`: Toggle Noclip (WASD, Shift to Boost, Scroll for Zoom)
